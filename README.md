# Farming Project

## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/farming.git
   cd farming

   ```

2. Install dependencies:
   npm install

3. Run the development server:
   npm run dev

4. Run tests:
   npm run test
